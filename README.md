# lost-found-chatbot

# 🔍 Lost & Found Chatbot System

A simple web application to report and find lost items with a chatbot assistant.

---

## 🚀 Live Demo

🌐 Website:  
https://lost-found-chatbot.onrender.com  

🤖 Chatbot:  
https://lost-found-chatbot.onrender.com/chat  

---

## 📌 Features

- User Registration & Login
- Create Lost & Found Posts
- Upload Images
- View Recent Posts
- Admin Panel
- Chatbot Support (ask questions)
- Responsive UI using Bootstrap

---

## 🛠️ Technologies Used

- Python (Flask)
- SQLite Database
- HTML, CSS, Bootstrap
- JavaScript (for chatbot)
- Gunicorn (for deployment)

## ⚙️ How to Run Locally (VS Code)

### Step 1: Clone Repository

```bash
git clone https://github.com/shubdacsnc-source/lost-found-chatbot.git
cd lost-found-chatbot
Step 2: Create Virtual Environment
python -m venv venv
venv\Scripts\activate   # Windows
Step 3: Install Requirements
pip install -r requirements.txt
Step 4: Run Application
python app.py
Step 5: Open in Browser
http://127.0.0.1:5000/
